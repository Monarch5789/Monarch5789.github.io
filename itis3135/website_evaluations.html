Website Evaluations

Spotless Agency

Spotless Agency is a website that appears dedicated to staging houses prior to sale, as well as a number of related services.

  • C: This page has strong contrast, but not in a harsh manner. The text is always distinct from the background, but not enough to cause discomfort
  • R: This page makes good use of repetition, as each new section keeps the same font sizes, colors, and headings.
  • A: The alignment on this page is also good, as nothing ends up overlaping or intruding on anything else.
  • P: Finally the proximity on this page is also good, as nothing feels cramped, and the content is not on the edge of the screen.

Overall, this website does follow the CRAP principles well, and looks pretty good.

The Big Ugly Website

The Big Ugly Website is a seemingly intentionally designed website that displays many violations of the CRAP principles.

  • C: This page struggles with some harsh contrast, most notably with the background, but also some highlighting.
  • R: This page also struggles with repetition, as many elements are different, and stand out in a negative light.
  • A: This page has some alignment issues with the images near the top, as they start to move into the other "column" of elements.
  • P: Finally, this page has proximity issues, with many elements being too close to the edge of the screen.

Overall, this website does not follow the CRAP principles very well, and is a good example of things to avoid.
